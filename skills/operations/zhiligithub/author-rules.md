# 作者名与草稿箱硬性规则（2026-05-21 确认）

## 作者名：「刘生」

**所有文章作者统一为「刘生」**，不要出现「卡兹克」。这条规则适用于：

| 位置 | 写法 |
|------|------|
| `publish_zhili.py` author 参数 | 固定传 `"刘生"` |
| HTML 文章底部 footer | `作者：刘生<br>来源：直隶按察使` |
| 飞书汇报消息 | 作者字段填「刘生」 |

**绝对禁止**：在任何生成的 HTML 内容、草稿 payload 或 API 参数中出现「卡兹克」。

## 草稿箱：不删除

**不要清空草稿箱**。草稿箱可以同时存在多篇草稿，直接上传封面 + 创建新草稿即可。

原来 `publish_zhili.py` 中可能存在「清空旧草稿」的步骤，**不要调用**，只调用：
1. `upload_thumb_material`（或 `add_material?type=thumb`）
2. `create_draft`（`draft/add`）

## 为什么这条规则重要

- 公众号账号已确定作者身份为「刘生」，混用作者名影响账号一致性
- 草稿箱有多篇草稿是正常状态，删除可能造成误操作
