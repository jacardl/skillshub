# 实体识别模式库

## L1：竞品关系强信号词（+2分）

### 中文
```
竞品、竞争对手、竞争品牌、直接竞争、对标品牌
替代品、可替代、可替代品、平替
同类产品、同品类、同一赛道
核心竞品、主要竞争、主要竞争对手
```
### 英文
```
competitor, rival, competing brand, direct competitor
alternative, substitute, replacement
same category, same market, same space
market rival, key competitor
```

### 组合短语
```
X 和 Y 是竞品、X 与 Y 竞争
X 的主要替代品、替代 X 的选择
X 的同类品牌、X 的竞争对手
与 X 同品类的品牌、与 X 竞争的品牌
```

## L3：句法结构模式（+2分）

以下句法结构直接表明竞品关系，识别后提取两侧实体：

### 强结构（直接竞品）

```regex
# "X 和 Y 是竞品" 类
(.{2,10})\s(和|与|同)\s(.{2,10})\s(是?竞品|竞争|竞争对手|对标)

# "X 的替代品" 类
(.{2,10})\s的\s(替代品|替代选择|可替代|替代)

# "与 X 竞争/对标的品牌" 类
与\s(.{2,10})\s(竞争|对标|同品类)的\s(品牌|产品|厂商|公司)

# "X 的同类品牌" 类
(.{2,10})\s的\s(同类|同类型|同赛道)\s(品牌|产品|厂商)
```

### 弱结构（间接竞品）

```regex
# "X 比 Y 更..." 类（需确认前后是品牌）
.{2,10}\s比\s.{2,10}\s(更|还|要)

# "除了 X，还有 Y" 类
除了\s(.{2,10})(，还有|还有|还有)\s(.{2,10})

# "X 也做/也有/也在做" 类
(.{2,10})(也|同样)\s(做|有|提供)
```

## L5：Target 品牌屏蔽词构建

从 KB 读取 Target 品牌元数据后，构建屏蔽词正则：

```
标准名 → 全大写/全小写/首字母大写变体
英文名 → 原文 + 常见缩写
别名 → 原词 + 拼音变体 + 打字错误容差（编辑距离≤2）
产品线 → 产品名也加入屏蔽（避免"XX产品"被识别为竞品）
```

## 特殊处理

### 中文品牌拼音识别
```
常见拼音映射规则：
- 取品牌名每个汉字首字母
- 允许声调省略（guoli = 果栗）
- 允许 ü→u, ie→i 等常见拼写变体
```

### 英文品牌大小写不敏感
```
全部转为小写后匹配
Apple = apple = APPLE
Google = GOOGLE = google
```

### 嵌套品牌名处理
```
"华为荣耀" ≠ "华为"（需区分父子品牌）
解决：精确匹配优先于包含匹配
```

---

## 识别优先级

```
1. 精确匹配已知别名库
2. 句法结构提取（最高可信）
3. 信号词 + 上下文综合判断
4. LLM 语义推断（兜底，质量最低）
```

---

## 误识别高风险模式

⚠️ 以下情况需特别排除：

```
1. "XX公司" ← 可能是母公司而非竞品
2. "XX平台" ← 可能是渠道而非竞品
3. "用户提到的XX" ← 上下文不足
4. "据XX报道" ← 是媒体不是竞品
5. "XX和YY" 出现在列表中但无关系词 ← 可能是无关并置
```
